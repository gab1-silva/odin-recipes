# odin-recipes

This project is about creating a small website of a collection of recipes and using git and github to save changes and publish it, with the objective of
refining my HTML skills and getting me used to the git workflow.

As I'm writing this, the project is finished, and I can say that I have learned not only the basic structure of a website, but also a lot about
organizing the file system of said website and about using git often to make my life a lot easier :).
